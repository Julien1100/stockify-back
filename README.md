# Stockify | Back-End

_This project is not open to contribution._

## Dependencies

- Backpack-core
- Bcrypt JS
- Body-parser
- CORS
- DotEnv
- Express
- JsonWebToken
- Mongoose
- Volleyball _(dev dependecy)_

## How to use

1. Clone this project
2. Install dependencies using `npm install`
3. Create a `.env`. file _(exemple in `exemple.env` file)_
